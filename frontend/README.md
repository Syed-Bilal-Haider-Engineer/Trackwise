# TrackWise Frontend

Web frontend built with React + Vite.

## Scripts

- `npm run dev` - Start local dev server
- `npm run build` - Build production assets
- `npm run preview` - Preview production build
- `npm run lint` - Lint source files

## Local Development

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the app
   ```bash
   npm run dev
   ```
